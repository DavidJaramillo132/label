<h1>

Representa el encabezado de una página. Tiene 6 niveles (h1 a h6).

<h1>Texto del encabezado</h1>
<title>

Define el título que aparece en la pestaña del navegador.

<title>Mi página web</title>
<body>

Contenedor de todo el contenido visible de la página.

<body>Contenido visible</body>
<a>

Crea un enlace a otra página o parte del documento.

<a href="https://ejemplo.com">Ir a ejemplo</a>
<img>

Muestra una imagen.

<img src="foto.jpg" alt="Descripción">
<input>

Campo para que el usuario ingrese datos.

<input type="text" placeholder="Escribe aquí">
<button>

Crea un botón interactivo.

<button>Clic aquí</button>
<div>

Contenedor genérico para agrupar elementos.

<div>Contenido agrupado</div>
<ul>

Lista no ordenada con viñetas.

<ul><li>Elemento</li></ul>
<ol>

Lista ordenada con números.

<ol><li>Elemento</li></ol>
<li>

Elemento de una lista.

<li>Elemento</li>
<main>

Contenido principal de la página.

<main>Contenido principal</main>
<code>

Fragmento de código.

<code>console.log("Hola");</code>
<section>

Define una sección del contenido.

<section>Contenido de sección</section>
<audio>

Reproductor de audio.

<audio controls><source src="audio.mp3" type="audio/mpeg"></audio>
<video>

Reproductor de video.

<video controls><source src="video.mp4" type="video/mp4"></video>
<strong>

Texto con importancia. Se muestra en negrita.

<strong>Importante</strong>
<mark>

Texto resaltado como si fuera con marcador.

<mark>Texto marcado</mark>
<hr>

Inserta una línea horizontal.

<hr>
<span>

Contenedor en línea para estilos en texto.

<span style="color:red;">Texto en rojo</span>
